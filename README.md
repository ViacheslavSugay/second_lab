## Python. Лабораторная работа № 2

### Основные команды (Easy)

- **ls** - список файлов и каталогов
  - Поддержка пути `path`
  - Опция `-l` для подробного отображения (права доступа, размер, дата изменения)

- **cd** - смена каталога
  - Поддержка `..` (на уровень выше)
  - Поддержка `~` (домашний каталог)

- **cat** - вывод содержимого файла

- **cp** - копирование файлов/каталогов
  - Опция `-r` для рекурсивного копирования каталогов

- **mv** - перемещение/переименование файлов/каталогов

- **rm** - удаление файлов/каталогов
  - Опция `-r` для рекурсивного удаления
  - Подтверждение для удаления каталогов
  - Защита от удаления корневого каталога

### Логирование

- Все команды и ошибки записываются в файл `shell.log`
- Формат: дата/время, команда, результат (успех/ошибка)
